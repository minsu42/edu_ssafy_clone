## 정리 기준

- CSV 표를 Markdown 표와 상세 항목으로 변환했습니다.
- Notion URL은 가독성을 위해 식별자만 남겼습니다.
- 기존 원본 파일은 삭제하지 않았습니다.
- 검사기준서 항목은 원본에 세부 CSV가 없어 TODO로 남겼습니다.
