# 1. 프로젝트 목표

다양한 프로젝트 환경에서도 일관된 코드 생성 품질을 확보하기 위해 멀티 에이전트 기반 하네스 프롬프팅 구조를 설계하고, 실제 서비스 클론 코딩을 통해 이를 검증한다. 단순 코드 생성이 아니라 요구사항 해석부터 설계, 구현, 테스트, 수정까지를 분리된 에이전트로 구성하고, 실패 케이스에서의 자동 복구와 반복 개선 흐름을 실험한다.
